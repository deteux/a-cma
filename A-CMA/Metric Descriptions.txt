numOps - Number Of Methods Per Class

setters - Number Of Setter Methods Per Class
getters - Number Of Getter Methods Per Class
staticness - Average Number Of Static Methods Per Class

nesting - Nesting Level Per Class

PACKAGE METRICS

numCls - Number Of Classes In Package
numInterf - Number Of Interfaces In Package
packagenesting - Nesting Level Of Package
numOpsCls - Number Of Methods In Package

INHERITANCE METRICS
iFImpl - Number Of Interfaces Implemented By Class
NOC - Number Of Children Per Class
numDesc - Number Of Descendants Per Class
numAnc - Number Of Ancestors Per Class

eC_Attr - Number Of External Uses Of Class As Attribute Type

Dep_In - Number Of Elements That Depend On Class (Use Class As Type)
Dep_Out - Number Of Elements Depended On By Class
NumAssEl_ssc* - Number Of Associated Elements In The Same Package
NumAssEl_nsb* - Number Of Associated Elements
